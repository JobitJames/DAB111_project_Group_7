Data Set Name: Canada Climate Dataset

Dataset Overview

Total Rows: 61
Total Columns: 8

Column Details

Numerical: x, y, STN_ID, ELEVATION, first_year, last_year
Categorical: STATION_NAME, PROV_STATE_TERR_CODE

Numerical Columns

x, y, STN_ID, ELEVATION, first_year, last_year
These columns contain quantitative values essential for spatial, temporal, and statistical analyses.

Categorical Columns

STATION_NAME, PROV_STATE_TERR_CODE
These columns describe qualitative aspects of the data, such as station names and locations.

x

Represents the longitude of the station. It helps identify the station's horizontal position on the Earth's surface.

y

Refers to the latitude of the station, indicating its vertical position on the globe.

STN_ID

A unique numeric identifier for each weather station. It ensures that each station can be distinctly referenced in the database.

ELEVATION

Indicates the height of the station above sea level, measured in meters. Elevation is vital in understanding how altitude influences climate observations.

first_year

Marks the year when the station began recording climate data, providing the start of its historical records.

last_year

Denotes the most recent year of data collection, indicating how current the data from the station is.

STATION_NAME

Specifies the name of the station, often based on its geographic location or local identifier.

PROV_STATE_TERR_CODE

Represents the code for the province, state, or territory where the station is situated, helping in categorizing and sorting the stations regionally.

Purpose

The dataset is essential for analyzing climate patterns across Canada, integrating geography, time, and regional factors.
